# Advance ToDo App

## 📌 Description
This is a simple project built using React. It helps users manage tasks with drag and drop functionality.

## 🚀 Features
- Add,Delete and edit tasks
- Drag & drop (To Do / In Progress / Done)
- Due Date Tracking
- Priority based task organiztion

## 🛠️ Tech Stack
- React
- JavaScript
- CSS

## 📂 Installation
1. Clone the repository:
git clone https://github.com/sharondillu/todoapp.git

2. Go to folder:
cd todoapp
3. Install dependencies:
npm install

4. Run the app:
npm run dev
